# Rick Astley Fan Site

Ласкаво просимо на фан-сайт, присвячений легендарному британському виконавцю Ріку Естлі!

## Про проект
Цей сайт був створений в рамках серії лабораторних робіт з дисциплін:
- **Веб-технології та веб-дизайн**
- **Управління проєктами та комунікації**

## Функціонал
- **Головна сторінка**: Загальна інформація, карта навігації.
- **Біографія**: Детальна таблиця з етапами кар'єри.
- **Галерея**: Фотографії виконавця.

## Технології
- HTML5 (Semantic tags)
- CSS3 (Flexbox, Gradients, Shadows, Transitions)
- Git & GitHub Pages
